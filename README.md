# Zootopia - Animal Information Web Generator

## Description

Zootopia is a web application that fetches animal information from the API Ninja API and generates a beautiful HTML webpage displaying details about different animals.

## Features

- ✅ Fetch animal data from API Ninja
- ✅ User input for animal name
- ✅ Generate HTML webpage with animal information
- ✅ Error handling for non-existent animals
- ✅ Secure API key management with .env
